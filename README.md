# ML Service Project

Ml service project for Glioma classification 

## Stack 

1. sqlite
2. redis 
3. dash 
4. FastApi 

## Installation 

```
git clone https://github.com/LongArya/ITMOMLServicePractice.git
cd ITMOMLServicePractice/
sudo docker compose up --build
```

navigate to http://localhost:10000/
